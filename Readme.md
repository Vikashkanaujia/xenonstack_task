
# XenonStack Hiring Task

It is a website that consists of 
1) Register Page
2) Login Page
3) HomePage  
4) Contact Us Page

## Tech Stack Used :
#### Frontend :
    1. HTML
    2. CSS
    3. JAVASCRIPT
#### Backend :
    1. NODE JS
    2. EXPRESS JS
#### Database
    1. PostgreSQL

 
## Run Locally

Clone the project

```bash
  git clone https://github.com/Vikashkanaujia/xenonstack_task.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
### SETUP PostgreSQL

Setup postgreSQL server with following Details

![Postgres Details](https://raw.githubusercontent.com/Vikashkanaujia/xenonstack_task/main/screenshorts/postgres_login.PNG)

Create databases :
    
    1. users
    2. contact_us
